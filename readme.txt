Currently, this project can only be implemented on a localhost
and can be accessed via this link http://127.0.0.1:5500/index.html

1.Install the required dependencies:
	pip install -r requirements.txt

2.Create a .env file and add your Supabase credentials:
	SUPABASE_URL=your_supabase_url
	SUPABASE_KEY=your_supabase_key

3.Start the backend server:
	uvicorn main:app --reload

4.Open your browser and go to:
	http://localhost:8000

5.Register or log in to start using the To-Do List application.
